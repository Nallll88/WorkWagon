Company Introduction

WorkWagon is a full-stack web application built using Next.js, Prisma, and PostgreSQL. It streamlines the hiring process by enabling companies to post jobs and candidates to apply efficiently.

Company Details

  • Technology Stack: Next.js, Prisma ORM, PostgreSQL
  • Primary Users: Employers and Job Seekers
  • Features: Job Listings, Employer Dashboard, User Authentication
